# StudyVIBE Accessibility

Last updated: 2026-04-08

StudyVIBE is committed to providing an accessible experience for all users.

## Supported Accessibility Features

- Dark Interface support
- Text size preference in app settings (Small / Default / Large)

## How to Use Accessibility Settings in StudyVIBE

1. Open `Settings` in the app.
2. Go to `Uygulama Tercihleri`.
3. Choose a theme and adjust text size with the slider.
